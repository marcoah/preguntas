# Sistema de Preguntas

El objetivo de esta aplicacion es crear una interfaz para que los asistentes a un evento puedan hacer preguntas a los ponentes.

La aplicacion requiere estar disponible en un subdomino relacionado al evento y los participantes deber√°n escribir su nombre, telefono y correo para hacer las preguntas.
